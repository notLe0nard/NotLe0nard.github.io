BedgeBot Commands:


#time

Sends the current time.


#help | #commands

Brings you here.


#gn | #guna

Start to sleep. The bot will send a message when you wake up with the time you spent sleeping.


#ping | #test

Simple command to just see if the bot is active.


#follows | #follower

A list of all accounts the given channel is following.


#vips

A list of all VIPs in the current channel.


#mods | #moderators

A list of all Mods in the current channel.


#founders

A list of all Founders in the current channel.


#color

Shows the color code of your Userame.
